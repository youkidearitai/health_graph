# Health Graph Application

集めたデータ、グラフにしようと思って作った。まだテスト段階です。

__DO NOT PRODUCTION USE__

## how to install

    $ virtualenv env
    $ source env/bin/activate
    $ pip install -r requirements.txt
    $ cd keep_health
    $ manage.py migrate
    $ manage.py runserver

Connect to http://127.0.0.1:8080
